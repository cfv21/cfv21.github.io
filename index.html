about me

i’m calvin, a soon-to-be computer science graduate from florida state university with a minor in mathematics. i’ve spent the last few summers immersed in real-world tech — first as a data analyst, then as an it intern — where i built automation scripts, optimized networks, and developed an appreciation for elegant problem-solving.

i’m passionate about python, large datasets, and building systems that make sense. recently, i’ve been working on a project called the climate change impact analyzer, where my team and i are using machine learning to make sense of environmental data from nasa and noaa. i’m writing custom modules, building data pipelines, and helping train models that identify trends and anomalies in global climate patterns.

alongside that, i’ve deployed and currently manage a self-hosted linux web server — setting up nginx, optimizing network configs, hardening security, and monitoring resources to serve content reliably to the public.

i like solving real problems, learning what works, and improving what doesn't. theres always a better way to write code, design systems, or solve challenges -- and i enjoy finding it.
